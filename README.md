# Lodemark.ai — Landing Page

An outstanding, brand-faithful homepage for **Lodemark AI** — *a clearer path to stable scale.*

Built from the official Lodemark.ai brand presentation: teal `#18D6C4` on deep black, the geometric "connected system" mark, and Nexa-style geometric type (Space Grotesk / Inter).

## Highlights

- ⚡ Single self-contained `index.html` — no build step
- 🎨 On-brand dark theme with neon-teal glow and the four-rotated-"L" logo mark
- 🖼️ Real brand imagery from the deck (AR-headset hero, billboard, team)
- 📱 Fully responsive — desktop hero + a dedicated mobile "app-screen" layout
- ✨ Scroll-reveal animations, animated stat counters, hover-spotlight cards, bento showcase
- 🔖 Favicon set generated from the brand mark

## Structure

```
index.html          # the full page (inline CSS + JS)
assets/             # images + favicons
  hero.jpg          # AR-headset hero visual
  texture.jpg       # brandmark on material
  sc-billboard.jpg  # "+50K deployments" billboard
  sc-team.jpg       # team collaboration
  favicon.*         # svg / ico / png / apple-touch-icon
```

## Run

Open `index.html` in any modern browser. Keep the `assets/` folder alongside it.
